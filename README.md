# 📚 AI Book Librarian

An AI-powered virtual librarian built using **Hugging Face Transformers**, **Gradio**, and **Python**. This application allows users to discover books, receive personalized recommendations, and interact with an intelligent AI assistant through natural language conversations.

## ✨ Features

- 📖 AI-powered book recommendations
- 🔍 Smart book search
- 💬 Interactive chatbot interface
- 🤖 Hugging Face language model integration
- 📚 Personalized reading suggestions
- ⚡ Fast and user-friendly Gradio interface

## 🛠️ Tech Stack

- Python 3.13
- Gradio
- Hugging Face Transformers
- Natural Language Processing (NLP)
